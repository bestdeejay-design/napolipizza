## 📋 СЛЕДУЮЩИЕ ШАГИ

### 1. Добавь удаленный репозиторий

```bash
git remote add origin https://github.com/ТВОЙ_НИК/ТВОЙ_РЕПОЗИТОРИЙ.git
```

Или через SSH:
```bash
git remote add origin git@github.com:ТВОЙ_НИК/ТВОЙ_РЕПОЗИТОРИЙ.git
```

### 2. Запуш проект

```bash
./deploy.sh
```

Или вручную:
```bash
git push -u origin main
```

### 3. Включи GitHub Pages

1. Зайди в репозиторий на GitHub
2. Settings → Pages
3. Source: Deploy from branch
4. Branch: main, Folder: / (root)
5. Save

### 4. Готово!

Через 1-2 минуты сайт будет доступен:
```
https://ТВОЙ_НИК.github.io/ТВОЙ_РЕПОЗИТОРИЙ/
```
