### Вариант 1: Python HTTP Server (Рекомендуется для разработки)

**macOS/Linux:**
```bash
chmod +x start.sh
./start.sh
```

**Windows:**
```bash
start.bat
```

Сайт откроется по адресу: **http://localhost:8000**

### Вариант 2: Docker (Требует запущенного Docker Desktop)

```bash
docker-compose up --build -d
```

Сайт откроется по адресу: **http://localhost:8080**

Проект готов к демонстрации заказчику. Для дальнейшей разработки:

4. При необходимости обновите цвета в `:root` CSS переменных
