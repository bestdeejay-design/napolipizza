## 📋 СЛЕДУЮЩИЕ ШАГИ

### 1. Включи GitHub Pages ⚙️

1. Зайди в репозиторий: **https://github.com/bestdeejay-design/napolipizza**
2. Перейди во вкладку **Settings**
3. В левом меню выбери **Pages**
4. В разделе **Source** выбери:
   - **Deploy from branch**
5. Настрой ветку и папку:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
6. Нажми **Save**

### 2. Подожди 1-2 минуты ⏳

GitHub Pages автоматически сгенерирует сайт.

### 3. Открой сайт 🌐

Сайт будет доступен по адресу:
```
https://bestdeejay-design.github.io/napolipizza/
```
