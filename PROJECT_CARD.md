## 👋 Приветствие

**Привет!** Я создал для тебя потрясающий сайт итальянского ресторана **Napoli Pizza** уровня Michelin ⭐⭐

## 🚀 Как посмотреть

### Вариант 1: Быстрый просмотр
**Сайт уже запущен!** Просто нажми кнопку **"preview"** справа → 

Откроется http://localhost:8000

### Вариант 2: Перезапуск
```bash
./start.sh        # macOS/Linux
start.bat         # Windows
```

### Вариант 3: Docker (когда заработает)
```bash
docker-compose up --build -d
```
