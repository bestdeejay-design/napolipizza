# Napoli Pizza - Исходные изображения

## Инструкции по изображениям

Для завершения сайта необходимо добавить следующие изображения в папку `src/images/`:

### Основные изображения:
1. **hero-pizza.jpg** - Главная фотография пиццы для hero-секции (1920x1080px)
2. **chef-cooking.jpg** - Шеф-повар готовит пиццу (600x800px)
3. **pizza-oven.jpg** - Дровяная печь для пиццы (400x500px)
4. **restaurant-interior.jpg** - Интерьер ресторана (800x600px)
5. **pizza-making.jpg** - Процесс приготовления пиццы (400x500px)
6. **wine-selection.jpg** - Винная карта/вина (400x500px)
7. **dessert.jpg** - Итальянские десерты (400x500px)
8. **chef-portrait.jpg** - Портрет шеф-повара (400x600px)

### Рекомендации по стилю:
- Теплые, аппетитные цвета
- Профессиональное освещение
- Итальянская эстетика
- Высокое разрешение (минимум 150 DPI)
- Формат: JPG или PNG

### Временное решение:
Пока изображения не добавлены, сайт будет использовать цветные заглушки.
