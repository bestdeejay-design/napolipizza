## 🎨 Дизайнерские решения

### Цветовая схема
- **#c41e3a** - Итальянский красный (акценты, кнопки)
- **#2c3e50** - Глубокий синий (заголовки)
- **#f39c12** - Золотой (детали)
- **#faf8f5** - Теплый белый (фон)

### Типографика
- **Playfair Display** - элегантные заголовки с засечками
- **Lato** - чистый текст без засечек

### Пространство
- Section padding: 100px
- Container max-width: 1200px
- Щедрые отступы между элементами

### Градиенты (вместо фото)
- Фиолетовые → Розовые
- Синие → Бирюзовые
- Оранжевые → Желтые
- Персиковые → Коралловые

**Решение:** SVG заглушки выглядят стильно и современно, не требуют поиска фотографий!
