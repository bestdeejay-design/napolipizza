# 🍕 Pizza Napoli - Структура проекта

## 📁 Два сайта в одном проекте

### 1. **Сайт для заказчика** (основной)
**Путь:** `/src/`  
**Контент:** Реальный контент с pizzanapolirsc.ru  
**Дизайн:** На основе оригинального сайта заказчика  
**Статус:** ✅ ГОТОВ К ПОКАЗУ

**Запустить:**
```bash
./start.sh
```
**URL:** http://localhost:8000

---

### 2. **Premium Concept** (альтернативный дизайн)
**Путь:** `/premium-concept/`  
**Дизайн:** Премиум версия в стиле Michelin ⭐⭐  
**Контент:** Вымышленный для демонстрации дизайна  
**Статус:** ✅ ГОТОВ КАК АЛЬТЕРНАТИВА

**Запустить:**
```bash
# Временно переместить файлы
mv src/* temp-backup/
mv premium-concept/* src/

# Запустить
./start.sh

# Вернуть обратно
mv src/* premium-concept/
mv temp-backup/* src/
rm -rf temp-backup/
```

**URL:** http://localhost:8000

## 🚀 Как работать с проектом

### Для показа заказчику

**Вариант 1: Сайт заказчика (рекомендуется)**
```bash
./start.sh
```
Открывается http://localhost:8000

**Вариант 2: Premium концепт (как альтернатива)**
1. Сделать backup текущих файлов
2. Переместить premium-concept в src
3. Показать заказчику
4. Вернуть всё обратно

### Для разработки

**Редактирование сайта заказчика:**
```bash
# Открыть файлы
code src/index.html
code src/styles/main.css
code src/scripts/main.js
```

**Редактирование Premium концепта:**
```bash
# Открыть файлы
code premium-concept/index.html
code premium-concept/styles/main.css
code premium-concept/scripts/main.js
```
