## 📋 ЧТО НУЖНО СДЕЛАТЬ ПРЯМО СЕЙЧАС

### Шаг 1: Добавь remote origin

```bash
git remote add origin https://github.com/ТВОЙ_НИК/pizzanapoli.git
```

**Где:**
- `ТВОЙ_НИК` - твой username на GitHub
- `pizzanapoli` - название репозитория (можешь изменить)

**Или через SSH:**
```bash
git remote add origin git@github.com:ТВОЙ_НИК/pizzanapoli.git
```

---

### Шаг 2: Запуш проект

```bash
git push -u origin main
```

**Используй скрипт:**
```bash
./deploy.sh
```

---

### Шаг 3: Включи GitHub Pages

1. Зайди в репозиторий на GitHub
2. **Settings** → вкладка **Pages**
3. **Source:** Deploy from branch
4. **Branch:** main
5. **Folder:** / (root)
6. Нажми **Save**

---

### Шаг 4: Готово!

Через 1-2 минуты сайт будет доступен:

```
https://ТВОЙ_НИК.github.io/pizzanapoli/
```
