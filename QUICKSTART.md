# 🍕 Napoli Pizza - Быстрый старт

## ✅ Сайт запущен!

**Откройте сайт:** Нажмите кнопку **preview** справа или перейдите на http://localhost:8000

---

## 📋 Что сделано

### ✨ Премиум дизайн итальянского ресторана Michelin уровня
- Современный одностраничный сайт (landing page)
- Адаптивная верстка (мобильные, планшеты, десктоп)
- Интерактивное меню с 5 категориями
- Форма бронирования столика
- Галерея с lightbox
- Плавные анимации и параллакс эффекты

### 🎨 Стиль
- Итальянская цветовая палитра (красный #c41e3a)
- Элегантная типографика (Playfair Display + Lato)
- Щедрые отступы, воздушный дизайн
- Градиентные placeholder вместо фото (стильно!)

### 🛠 Технологии
- Чистый HTML5 + CSS3 + JavaScript (ES6+)
- Без библиотек и зависимостей
- Docker + Nginx для production
- Python HTTP server для разработки

---

## 🚀 Управление

### Запустить сервер (если остановлен)
```bash
./start.sh          # macOS/Linux
start.bat           # Windows
```

### Остановить сервер
В терминаале нажмите `Ctrl + C`

### Перезапустить Docker (когда заработает)
```bash
docker-compose down
docker-compose up --build -d
```

## 🎯 Что можно улучшить

### 1. Добавить изображения (опционально)
Поместите в `src/images/`:
- hero-pizza.jpg (1920x1080)
- chef-cooking.jpg (600x800)
- pizza-oven.jpg (400x500)
- restaurant-interior.jpg (800x600)
- pizza-making.jpg (400x500)
- wine-selection.jpg (400x500)
- dessert.jpg (400x500)
- chef-portrait.jpg (400x600)

**Или оставьте градиентные заглушки - они выглядят стильно!**

### 2. Обновить контент
- Цены в меню (сейчас EUR €)
- Контактные данные
- Ссылки на соцсети

### 3. Настроить backend
Форма бронирования показывает alert. Для реальной работы:
- Отправка на email
- Интеграция с CRM
- Telegram бот

### 4. Deploy на production
```bash
# GitHub Pages
git add .
git commit -m "Napoli Pizza website"
git push origin main

# Или используйте Docker
docker-compose up --build -d
```
